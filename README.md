# Who2Global GitHub Defaults

This repository contains organisation-wide GitHub defaults for Who2Global.

It provides shared templates and guidance that can be inherited by other repositories in the Who2Global GitHub organisation.

## Purpose

The purpose of this repository is to keep common GitHub working practices consistent across Who2Global repositories.

This includes shared templates for issues, pull requests, contribution guidance and support information.

## What belongs here

This repository may include:

* Pull request templates
* Issue templates
* Contribution guidance
* Security reporting guidance
* Support information
* General organisation-level GitHub defaults

## What does not belong here

This repository must not contain:

* Secrets, passwords, tokens or certificates
* Private AWS account details
* Customer-sensitive information
* Internal-only architecture
* Deployment credentials
* Commercial or confidential documentation

Private standards, internal workflows and technical governance should be stored in private repositories owned by the Who2Global organisation.

## Repository status

This repository is part of the Who2Global GitHub foundation.

It should remain simple, safe and suitable for public visibility.
